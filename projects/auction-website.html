AuctionX: Auction Website

A full-featured real-time auction platform built with vanilla HTML5, CSS3, and JavaScript, featuring live bidding with countdown timers, role-based dashboards, multi-session account switching, and a complete payment flow with timeout handling.
Includes dynamic product rendering, search with debouncing, category filtering, and persistent data management via localStorage/sessionStorage. The frontend uses a mobile-first responsive design with custom modals, toasts, and smooth animations.
Fully automated deployment pipeline using GitHub Actions for CI/CD, ensuring every push triggers a build and goes live instantly. Hosted on GitHub Pages with zero downtime and free SSL. Version controlled and managed entirely on GitHub.
